## [v1.0] - 2019.11.10
* 1.基于AT32F4xx标准外设库(1.2.0版)，移植了全部的ArduinoAPI

## [v1.1] - 2019.11.14
* 1.更新timer库，更精确的定时频率

## [v1.2] - 2019.11.23
* 1.修复定时器库频率不对的BUG

## [v1.3] - 2019.12.1
* 1.更新timer库，使用独立的开根号运算函数，不依赖math.h库

* 1.工程框架修改，工程文件与代码文件分离

* 3.添加了GNU支持
